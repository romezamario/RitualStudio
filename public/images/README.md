# Estructura de imágenes

Esta carpeta centraliza todos los assets estáticos del sitio.

## Árbol sugerido

- `home/hero`: imágenes principales de portada.
- `home/highlights`: imágenes de bloques destacados en inicio.
- `home/testimonials`: fotos de clientes para testimonios.
- `home/banners`: piezas promocionales de campañas.

- `arreglos/catalogo`: fotos principales de arreglos para venta.
- `arreglos/temporada`: colecciones por temporada.
- `arreglos/detalles`: close-ups de flores y acabados.
- `arreglos/thumbs`: miniaturas de tarjetas/listados.

- `nosotros/equipo`: retratos del equipo.
- `nosotros/taller`: fotos del espacio de trabajo.
- `nosotros/proceso`: pasos del proceso creativo.
- `nosotros/historia`: imágenes de trayectoria de marca.

- `eventos/bodas`: portafolio para bodas.
- `eventos/corporativos`: eventos empresariales.
- `eventos/sociales`: celebraciones sociales.
- `eventos/montajes`: instalaciones y montaje en sitio.

- `custom/inspiracion`: referencias visuales para clientes.
- `custom/briefs`: imágenes asociadas a briefs activos.
- `custom/referencias-clientes`: fotos enviadas por clientes.

- `branding/logo`: logos oficiales (claro/oscuro/monocromo).
- `branding/iconos`: íconos de marca y UI.
- `branding/paleta`: recursos visuales de color y guía.

- `blog/covers`: portadas de artículos.
- `blog/galerias`: fotos internas de posts.

- `testimonios/clientes`: retratos o fotos de clientes.
- `testimonios/casos-exito`: antes/después y casos documentados.

- `seo/og`: imágenes Open Graph por página.
- `seo/share`: versiones para compartir en redes.
- `seo/favicons`: favicons y variantes de app icons.

- `placeholders`: imágenes temporales mientras llega fotografía final.

## Convenciones recomendadas

- Formato: `webp` o `jpg` para foto; `png` para transparencias.
- Nombres: minúsculas, guiones medios y sin espacios.
- Evitar duplicados: reutilizar `thumbs` para vistas pequeñas.
